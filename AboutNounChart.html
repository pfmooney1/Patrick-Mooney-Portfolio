Monopoly Bank Accounts

The Idea

What if you could play Monopoly without touching the bills?
The slowest part of playing Monopoly is handling the money and crunching numbers. While it's fantastic when someone lands on your monopoly with four houses on it, 90% of the time it is counting and passing back and forth less than $95.
What if there was a faster way? What if there were monopoly bank accounts?

The Approach

I made a table tall enough to fit up to 10 players and made it wide enough to fit enough actions to play the board game without touching the paper bills. I had just learned how to create and use Modals and I applied them to the project as well, creating the Start and Repair screens. I also used SVG icons instead of PNG or JPGs to take advantage of their speed and scalability. I wanted to make it customizable when playing so I added the ability to change the font, color theme, and the player names.

The Challenge

This project presented two challenges.
First, as I coded the modals it got increasingly hard to stop bugs from popping up, especially in the repairs modals. I would edit a modal and not find the changes taking place, only to realize that I was coding in the wrong modal.
Second, the dropdown select element that I had coded for the alternate themes/fonts looked ugly and was unappealing to use.

The Solution

I solved the modals challenge with clear commenting above each modal in the HTML file. This made it easy to know for sure which modal I was editing and how it functioned in relation to the rest of the modals. An ounce of prevention is worth a pound of cure.
Second, I replaced my dropdown/select elements with toggle tokens coded in vanilla HTML/CSS/JS. I liked these because it made it clear which one was selected, it gave the user a preview of what they were choosing, and they looked significantly better.

My Favorite Part

The piece of code I am most proud of is the JavaScript function the takes the number of houses and hotels a player has and charges them accordingly based on whether they landed on Chance or Community Chest. Stopping a game of monopoly to crunch those numbers was always annoying, and I love how a problem like that can be solved in so few lines of code.

Here's the code: 

                    
                        // Calculates the repair bill
                        function calculateRepairBill(a) {
                          if (a === 'communityChest') {
                            communityChestHouses = document.getElementById("communityChestHouses").value;
                            communityChestHotels = document.getElementById("communityChestHotels").value;
                            communityChestHouses = Number(communityChestHouses);
                            communityChestHotels = Number(communityChestHotels);
                            housesRepairCost = communityChestHouses * 40;
                            hotelsRepairCost = communityChestHotels * 115;
                            repairBillTotal = housesRepairCost + hotelsRepairCost;
                            document.getElementById("repairsTotal").innerHTML = "$" + repairBillTotal;
                            document.getElementById("repairsCommunityChest1").style.display = "none";
                            document.getElementById("repairsChance1").style.display = "none";
                            document.getElementById("repairsFinalBill").style.display = "block";
                          } else if (a === "chance") {
                            chanceHouses = document.getElementById("chanceHouses").value;
                            chanceHotels = document.getElementById("chanceHotels").value;
                            chanceHouses = Number(chanceHouses);
                            chanceHotels = Number(chanceHotels);
                            housesRepairCost = chanceHouses * 25;
                            hotelsRepairCost = chanceHotels * 100;
                            repairBillTotal = housesRepairCost + hotelsRepairCost;
                            document.getElementById("repairsTotal").innerHTML = "$" + repairBillTotal;
                            document.getElementById("repairsCommunityChest1").style.display = "none";
                            document.getElementById("repairsChance1").style.display = "none";
                            document.getElementById("repairsFinalBill").style.display = "block";
                          }
                        }
