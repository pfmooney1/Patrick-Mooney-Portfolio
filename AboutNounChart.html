Noun Chart Generator

The Idea

The most time consuming part of teaching (aside from grading) is creating answer keys. This program solves that problem.
Before, you would need to turn your back to thirty hyper 6th graders to write on the board and hope everyone behaves while you write out all the forms. Now you can just press a few buttons and then it's on the board! Javascript has done all the heavy lifting for you.
Not only does it allow teachers to keep better tabs on their students, but it also saves valuable class and preparation time!

The Approach

This is my most Javascript-heavy project to date. I wrote the program to take a single Latin input (for example rex, regis, m) and parse it correctly no matter how the user separates the parts. Next, it identifies the noun's declension, which endings it should use, and the base. Next it puts it all together correctly in each of the twelve forms and displays them on the page.
The english half was easier. I take the user's input and run it through a function that checks to see if it is an irregular plural (ox -> oxen), then returns the proper plural form.
Additional features include the ability to quickly throw common irregular charts, random nouns, and blank charts on the board. For good measure, I even added a dark mode.

My Favorite Part

Having taught Latin for almost 5 years, it has been fascinating to create programs to do things that I teach my students how to do every day. Teaching my students how to identify the declension of any given Latin noun is one of those things. It's funny to see it in Javascript after all of these years.
Sometimes I wish it were that easy for my 6th graders to get it.

Here's the code: 

                    
function computeDeclension(genitiveSingularInput) {
    if (genitiveSingularInput.endsWith("ae")) {
        return "1st";
    }
    else if (genitiveSingularInput.endsWith("is")) {
        return "3rd";
    }
    else if (genitiveSingularInput.endsWith("us")) {
        return "4th";
    }
    else if (genitiveSingularInput.endsWith("ei")) {
        return "5th";
    }
    else if (genitiveSingularInput.endsWith("i")) {
        return "2nd";
    }
    else {
        alert("ERROR! Declension not figured out!");
    }
}
